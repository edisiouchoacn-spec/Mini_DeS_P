# Notas sobre Branches

# Branches criadas neste projeto

- 'docs/atualiza-readme' — documentação do repositório
- 'feat/hello-world' — adição do mini-projeto
- 'fix/versao-a' — versão A do hello-world em português
- 'fix/versao-b' — versão B do hello-world em inglês

# O que aprendi

Branches permitem trabalhar em paralelo sem afetar a main. Cada branch tem um propósito claro e é mergeada via Pull Request quando o trabalho está pronto.
